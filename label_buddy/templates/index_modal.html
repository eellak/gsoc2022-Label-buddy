Projects info
Projects shown in this page are projects which you are involved to. You must be manager, annotator or reviewer for the project in order to enter it.

If you want to add a new prediction model meke sure you follow the steps below:

  • Give a unique name to your project.
  • The output labels must be comma separated.
  • Upload the entire model file (".h5" for Keras / ".pt" or ".pth" for TensorFlow).
  • Make sure that the final output of the model is a list of lists containing starting time, ending time and the predicted label for each segment. (i.e. [[0.0, 8.0, 'music'], [2.967, 8.0, 'speech']])
  • Upload the dataset for the model to be evaluated in.
  • Give the best evaluation precentage so far.
